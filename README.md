# 🏏 Sportify — Find & Join Cricket Matches Instantly

Sportify is a mobile-first web app that helps players find, join, and organize local cricket matches in seconds.

Built to solve a real problem:

> Finding players for a match is frustrating — people cancel, groups are inactive, and plans fail.

Sportify makes it simple:
👉 Open app → Join match → Play

---

## 🚀 Features

### 🔍 Find Matches

* View nearby cricket matches instantly
* See match details: time, location, players

### ⚡ Quick Join

* Join matches with one tap
* No complex setup or friction

### 🧲 Match Status System

* "Looking for players" → match forming
* "Confirmed match" → ready to play

### 🔥 Urgency Indicators

* "Only X spots left"
* "Starting soon"

### 🔗 Invite Players

* Share match link with friends
* Fill matches faster

### 📍 Location-Based (Gujrat Focus)

* Currently focused on Gujrat, Pakistan
* Designed for hyper-local sports communities

---

## 🧠 Problem It Solves

* Players can’t find enough people
* Matches get canceled last minute
* WhatsApp groups are unreliable

---

## 💡 Solution

A simple platform where:

* Matches are visible instantly
